>>	BDD

Definición -> Base de datos tradicional, dividida en diferentes partes físicamente dispersas y que es transparente para el usuario desde cualquier nodo.

- DDBMS(Dsitribuited Database Managment System) rige el almacenamiento y procesamiento de datos y se distribuyen en varios sitios(Rob Peter 2004)

___________________________________________________________

>>	Componentes del sistema de Administración de Base de Datos distribuidas

-> Estaciones de trabajo(sitios y nodos)

-> Componentes de software y hardware(baja,media o alta gama)

-> Medios de comunicación(la red)

-> El procesador de transacciones(Sistema Gestor de BD)(el administrador de BD hace esto)

-> El procesador de datos(Servidor se debe configurar)

___________________________________________________________

>>	Características de un SMBD BDD

1. Recibe la solicitud (usuario)
2. Valida,analiza y dscompone la solicitud(Op matemáticas,logicas o ambas) -> Procesador de datos
3. Descompone la solicitud en varias operaciones I/O de disco 
4. Busca, localiza, lee y valida ls datos -> Procesador de transacciones

*5.- Garantiza la consistencia, la seguirdad y la integridad. -> se hace en el medio de comunicacion.

6.- Valida los datos de conformidad con las condiciones. -> En las Querys en el gestor de transaciones
7.- Presenta los datos em el formato requerido -> Lenguaje SQL
8.- Todas las actividades son transparentes para el usuario.

>>	Ventajas y desventajas de DDBMS

Ventajas

1. Los datos se localizan cerca del sitio de "mayor demanda"
2. Acceso mas rápido a los datos
3. Procesamiento mas rápido de los datos.
4. Facilita el crecimiento
5. Comunicaciones mejoradas
6. Costos de operación reducidos
7. Interface de usuario fácil de usar
8. Menos peligro de falla en un solo punto
9. Independencia del procesador 

Desventajas

1. El rendimiento puede afectarse por la carga de trabajo
2. La confiabilidad de los sistemas disribuidos, por la complejidad de los componentes: ordenadores de red , almacenamiento, transacciones , replicación, etc.
3.- Por mayor complejidad ,los gastos de construcción.
